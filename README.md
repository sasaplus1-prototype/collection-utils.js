# collection-utils.js

utilities for collection object

## Installation

```sh
$ npm install sasaplus1-prototype/collection-utils.js
```

## Usage

via `require()`

```js
var collectionUtils = require('collection-utils');
```

via `<script>`

```html
<script src="collection-utils.min.js"></script>
```
